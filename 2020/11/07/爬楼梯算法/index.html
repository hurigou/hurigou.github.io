爬楼梯

解题过程

题目

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。
每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？
注意：给定 n 是一个正整数。
示例 1：
输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1 阶 + 1 阶
2 阶
示例 2：
输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1 阶 + 1 阶 + 1 阶
1 阶 + 2 阶
2 阶 + 1 阶
题目

解题过程

开始我是先把算法都列举出来，比如5阶时：1与2的结合就有7种，1+1…+1有一种，加起来就有8种。我本来想找出1+2结合的规律，但是我能力有限，我改用找总数的规律，我发现阶梯1、2、3、4、5、6
对应的的总数，分别为1、2、3、5、8、13.刚好符合斐波那契数列，也就是前面两个数相加等于后一位数所以我写的代码如下。

代码

#include”stdio.h”
void main()
{
int i,n;
int a[10];
printf(“输入楼梯阶层：\n”);
scanf(“%d”,&n);
a[0]=1;
a[1]=1;
for(i=2;i<=n;i++)
{
a[i]=a[i-1]+a[i-2];
}
printf(“一共有%d种方法走上该楼梯\n”,a[i-1]);
}
